Architecture

LCB Cinema는 전통적인 Spring MVC WAR 구조를 기반으로 한 영화관 서비스입니다. DispatcherServlet, XML 설정, JSP ViewResolver, MyBatis mapper XML, WebSocket handler를 중심으로 구성되어 있습니다.

Request Flow

Browser
  -> CharacterEncodingFilter
  -> DispatcherServlet
  -> Interceptor
  -> Controller
  -> Service
  -> MyBatis Mapper
  -> MySQL
  -> JSP View

Portfolio Layer

GitHub Pages의 index.html은 서버 없이 전체 서비스 흐름을 보여주는 인터랙티브 시뮬레이터입니다. Spring MVC 소스는 서버 구현 근거로 보관하고, 공개 데모는 빠르게 검토 가능한 제품 화면으로 제공합니다.
