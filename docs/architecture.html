Architecture

LCB는 Spring Boot가 아니라 전통적인 Spring MVC WAR 구조입니다. DispatcherServlet, XML 설정, JSP ViewResolver, MyBatis mapper XML을 중심으로 구성되어 있습니다.

Request Flow

Browser
  -> web.xml CharacterEncodingFilter
  -> DispatcherServlet
  -> interceptor-context.xml
  -> Controller
  -> Service
  -> MyBatis Mapper
  -> MySQL
  -> JSP View

Layering

Modernization Notes

포트폴리오 단계에서는 원본 구조를 유지하고 빌드와 보안을 먼저 정상화했습니다. 다음 단계에서는 SQL migration 정리, Spring Security 도입, 입력 검증, 테스트 커버리지, JSP 중복 레이아웃 정리가 필요합니다.
